## BleedStream DEDMASK 🚨

A protocol-level standalone URL engineering utility designed for deep credential pretexting and domain masking. It leverages native browser URL parsing mechanics (RFC 3986 UserInfo specifications) to visually mask destination URLs behind highly trusted target domains entirely offline or online.

Developed by node: **Unknownx007**  
Presented by **DEDSEC Community*

---

## ⚡ Key Features
* 🛡️ **Key words independency:** User can add it's own key words into masked URL.
* ⚙️ **Local Network Resilience Fallback:** 100% stable execution that slips smoothly into localized offline tracking profiles if external DNS or internet adapters drop out.
* 💬 **User friendly:** Dynamic User friendly interface which makes eazy.
---

## 🛠️ Installation & Setup

Execute these commands inside your Kali Linux terminal workspace to deploy the architecture:

```bash
# Clone the repository workspace profiles
git clone https://github.com/Unknownx007/DEDMASK

# Step into the operational target directory
cd DEDMASK

# Initialize and verify script compilation permissions
chmod +x dedmask.py

# Boot the tactical console framework natively
python3 dedmask.py
```
